# Upgrades

Vous pouvez également upgrade votre vile via le `/tlevels`  de l'argent disponible dans la banque de ville, ainsi qu'un nombre de claims requis vous est demander:

<figure><img src="../.gitbook/assets/image (2) (1).png" alt="" width="399"><figcaption></figcaption></figure>

Améliorer votre ville vous octroi des claims supplémentaire, des altacoins pour le maire ( voir [gagner-des-altacoins.md](../introduction/gagner-des-altacoins.md "mention")) ainsi qu'un multiplicateur de ressource.
