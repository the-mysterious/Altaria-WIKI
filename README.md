### 🔄 Synchronisation GitHub

Ce Wiki est **synchronisé automatiquement avec GitHub** :

* Chaque fichier `.md` correspond à une page GitBook
* Les mises à jour passent par des commits GitHub
* Les Pull Requests permettent de proposer des améliorations

Les contributions sont les bienvenues 🙌\

1. Fork le dépôt

3. Modifie ou ajoute un fichier `.md`
4. Vérifie le rendu sur GitBook
5. Ouvre une Pull Request détaillée
