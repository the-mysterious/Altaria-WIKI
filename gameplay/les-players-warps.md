# Les players-warps

La player warps (pw) sont des points de TP créés par des joueurs afin de rendre un lieu accessible au reste du serveur

Ils vous permettent de partager vos farm, base, build, shop (liste non exhaustive) facilement avec la communauté.

Selon les paramètres définis, un pw peut être en libre accès (gratuit) ou moyennant une contribution (payant avec de l'argent IG).



Les pw sont accessible via la commande: `/pw`&#x20;

<mark style="color:$danger;">**Les pw des autres joueurs | vos warp**</mark>\

**En passant votre souris sur un pw vous pouvez voir divers information comme:**

* **son nom**
* **le owner**
* **c'est CO**
* **son nombre de visites**
* **ect...**

## Comment créer un pw?

\
Pour ce faire, rien de plus simple : taper dans votre chat la commande suivante :\
`/pw create`

<figure><img src="../.gitbook/assets/image (2).png" alt="" width="301"><figcaption></figcaption></figure>

## Le menu

<figure><img src="../.gitbook/assets/image (3).png" alt="" width="380"><figcaption></figcaption></figure>

Lorsque vous exécutez la commande /pw -> My warps ->, vous cliquez sur votre le menu ci-dessus s'affiche, voyant en détail ce qu'il fait.

#### <mark style="color:$success;">**Ender pearl:**</mark>

Elle vous permet de vous téléporter à votre pw pratique non?

#### <mark style="color:orange;">**Le livre:**</mark>&#x20;

Vous permet de donner une description à votre pw

<figure><img src="../.gitbook/assets/image (4).png" alt="" width="563"><figcaption></figcaption></figure>

#### <mark style="color:$danger;">**La porte:**</mark>

<figure><img src="../.gitbook/assets/image (5).png" alt="" width="470"><figcaption></figcaption></figure>

Vous permettez de changer le statut du lock de votre pw. À noter que :

* un-locked : il est ouvert au public
* &#x20;locked: seulement vous pouvez y aller  <br>

#### <mark style="color:$primary;">**Le coffre:**</mark>

Il vous permet de choisir la catégorie de votre pw :

<figure><img src="../.gitbook/assets/image (22).png" alt="" width="209"><figcaption></figcaption></figure>

#### <mark style="color:$info;">**L'étoile du nether:**</mark>

Vous permet de bannir un joueur de votre pw.\
Pour se faire cliquer dessus et entrer son pseudo dans le chat IG :

<figure><img src="../.gitbook/assets/image (23).png" alt="" width="516"><figcaption></figcaption></figure>

#### <mark style="color:purple;">**Le cadre**</mark>

Il vous permet de changer l'icône de votre pw.

{% hint style="success" %}
Assurez-vous d'avoir l'item dans votre main avant de cliquer dessus
{% endhint %}

#### <mark style="color:$success;">Le name-tags:</mark>

Il vous permet de renommer votre pw facilement, pour se faire cliquer dessus et entrer le nouveau nom dans le chat IG

<figure><img src="../.gitbook/assets/image (24).png" alt=""><figcaption></figcaption></figure>

#### <mark style="color:orange;">**Le barrière bloc:**</mark>

Il vous permet de supprimer votre pw.

{% hint style="danger" %}
Attention, cette action est irréversible&#x20;
{% endhint %}

#### <mark style="color:$danger;">**La player head:**</mark>

Elle vous permet de transférer le ownership de votre pw

{% hint style="danger" %}
Attention, cette action est irréversible&#x20;
{% endhint %}
